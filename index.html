FREME Community

About this page

This is the landing page of the FREME community. FREME is a framework for multilingual and semantic enrichment of digital content. The framework has been developed in the context of the EU project  FREME (February 2016 - January 2017) and has found adoption beyond the project consortium. The goal of this page is to serve as an entry point for the emerging community.

Central resources

  • Homepage of the FREME EU project. Provides information about the project setup, motivating business cases, outreach activities, project results etc.
  • Framework documentation page. Provides a lot of information how to use the framework and (interactive) online demos.
  • GitHub framework repository. Contain the source code of the project, above documentation and more.
  • FREME community mailing list: a public list for discussions around the framework - feel free to join!

Activities / discussion topics

This is for now a placeholder section to list topics being discussed in the group. Once there is more information, we will link it from here.

  • Development of new e-Services, e.g. sentiment analysis.
  • Enhancement of XML support for various XML formats.

  • Creation of FREME empowered applications.
  • Deployment of FREME in (EU or other) R&D projects. The German Digital Curation Technologies has deployed the framework for the development of other e-Services. More information is available in the DKT GitHub repository.
